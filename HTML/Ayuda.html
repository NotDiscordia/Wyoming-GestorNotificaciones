Descripción de los íconos

Cerrar sesión

Se redireccionará a la página de inicio de sesión donde tendrá que ingresar sus datos nuevamente.

Agregar nueva noticia

Se redireccionará a la página de registro de noticias donde se tendrán que ingresar los datos correspondientes para dar de alta una noticia.

Agregar nuevo usuario

Se redireccionará a la página de registros de usuarios donde se dará de alta a un usuario que pueda acceder al sistema y también cederle permisos para que pueda registrar más usuarios.

Modificar

Se redireccionará a la página de registro de Usuario/Noticia solo de los datos de lo que se desea modificar en sus formularios correspondientes para facilitar al usuario hacer modificaciones.

Eliminar

En la gestión se mostrarán nuevos botones para poder eliminar los registros de Usuarios/Noticias en el sistema.

Deshacer

En gestión podrás deshacer los cambios que se hayan realizado a los registros, en este caso si se modificó algún nombre o título por accidente, este botón lo va a deshacer.

Rehacer

En gestión podrás rehacer los cambios que se hayan realizado a los registros, en este caso si accidentalmente se deshizo algún cambio, este botón lo restablecerá.

Búsqueda

En gestión se generará un cuadro de texto donde se escribirá el elemento a buscar en las tablas de consulta.

Ordenar

Sirve para ordenar los contenidos de las consultas a gusto del usuario, siendo que se ordenen por orden alfabético, por registro u otras opciones.
